# AI Assistant

A terminal-based AI assistant built with Python and Groq API.
Supports multi-turn conversations with a customizable system prompt.

## Tech Used
- Python
- Groq API (llama-3.3-70b)
- python-dotenv

## How to Run
1. Clone the repo
2. Install dependencies: `pip install groq python-dotenv`
3. Create a `.env` file with your Groq API key: `GROQ_API_KEY=your_key`
4. Run: `python assistant.py`
